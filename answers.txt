Task 1
The value in r3 after the loop is done is -1 in signed decimal and hexadecimal is ffffffff.

Task 2
The address of COUNTER in hexadecimal is 00001020. 

Task 3
Changing the value of LED_BITS in memory changes the initial pattern of the LED's while changing DELAY_TIME changes the speed of the LED rotation; larger delay values make it slower, and smaller values speed it up.
